# Your Energy

A Single Page Application for exploring fitness exercises, managing favorites, and staying motivated.

## Features

- Browse exercises by body parts, muscles, and equipment
- Keyword search with real-time filtering
- Save exercises to favorites (persisted in LocalStorage)
- Rate and review exercises
- Daily motivational quote (cached in LocalStorage)
- Responsive layout: 320px, 768px, 1440px

## Tech Stack

- Vanilla JavaScript (ES6+ modules)
- Vite 5 build tool
- CSS3 with custom properties, mobile-first approach
- Axios for HTTP requests
- Notyf for notifications
- DM Sans font (self-hosted WOFF2)
- AVIF/JPEG images with `<picture>` responsive elements

## Getting Started

```bash
npm install
npm run dev
npm run build
```
